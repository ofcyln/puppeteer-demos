## How to run scripts?

Install dependencies
```
npm i
```

Execute scripts with
```
node fps.js
cd whatsapp && node whatsapp.js
```

Run tests with
```
npx jest dom-snapshot.spec.js
cd budget && npx jest assert-metrics.spec.js
```
